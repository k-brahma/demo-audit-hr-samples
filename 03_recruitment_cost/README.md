# 03 採用コスト可視化ツール

## 概要

求人媒体ごとの応募数・採用数・費用データからCPA（Cost Per Acquisition：一人当たり採用コスト）を自動計算し、棒グラフで媒体比較を可視化するツール。

## 起動

```powershell
cd 03_recruitment_cost
python gui.py
```

## 使い方

1. 「CSVを開く」で `data/recruitment.csv` を選択
2. 上部サマリーカードに総費用・平均CPA・全体採用率が表示される
3. 左テーブルに媒体別の費用・CPA・採用率が一覧表示（CPA最安値を緑ハイライト）
4. 右グラフで媒体別CPAを棒グラフ比較（最安値が緑、それ以外が青）
5. 「結果を保存」で `results/recruitment_report.csv` に出力

## サマリーカードの指標

| 指標 | 説明 |
|------|------|
| 総応募数 | 全媒体合計の応募者数 |
| 総採用数 | 全媒体合計の採用者数 |
| 総費用(万円) | 全媒体合計の採用費用 |
| 平均CPA(万円) | 総費用 ÷ 総採用数 |
| 全体採用率(%) | 総採用数 ÷ 総応募数 × 100 |

## ファイル構成

```
03_recruitment_cost/
├── main.py          # CPA計算・採用率計算・サマリー集計ロジック
├── gui.py           # Tkinter GUI（サマリーカード + Treeview + matplotlib）
├── data/
│   └── recruitment.csv   # サンプル採用データ（8媒体）
└── results/
    └── recruitment_report.csv   # CPA計算済み出力
```

## サンプルデータ仕様

| 列名 | 内容 |
|------|------|
| 媒体名 | 求人媒体名（例: Indeed） |
| 応募数 | 応募者数 |
| 採用数 | 採用者数 |
| 費用 | 媒体利用費用（円） |

> ハローワーク（費用=0）も含まれており、CPA=0の媒体は比較対象から除外してグラフ描画する。

## 主要ロジック（main.py）

| 関数 | 内容 |
|------|------|
| `load_recruitment(filepath)` | CSV読み込み・数値型変換 |
| `calculate_cpa(df)` | CPA・採用率・万円換算列を追加 |
| `get_summary(df)` | 合計・平均サマリー辞書を返す |
| `analyze(filepath)` | ロードとCPA計算を統合 |

## 面接でのアピールポイント

- **採用担当経験の直接応用：** 「どの媒体が費用対効果が高いか？」という採用実務の問いをコードで答えた
- `pandas` の `apply` でCPA計算を行ベクトル処理していることを示せる
- Excelのピボットテーブルで行っていた分析を自動化した例として語れる
- 「コスト0のハローワーク」のようなエッジケースへの対処を説明できる
