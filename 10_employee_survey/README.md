# 10 従業員サーベイ 結果集計・分析ツール

## 概要

従業員エンゲージメントサーベイのCSVを読み込み、設問別の平均スコアと部署別比較をグラフで可視化するツール。棒グラフとレーダーチャートの2モードを切り替えて表示できる。

## 起動

```powershell
cd 10_employee_survey
python gui.py
```

## 使い方

1. 「CSVを開く」で `data/survey.csv` を選択
2. 上部に総合平均スコア（5点満点）と回答者数が表示される
3. 左テーブルに設問別の平均スコアが降順で表示（最高値=緑・最低値=赤）
4. 「表示モード」で「設問別（全体）」↔「部署別比較（レーダー）」を切り替え
5. 「結果を保存」で部署別集計を `results/survey_report.csv` に出力

## 表示モード

### 設問別（全体）棒グラフ
全回答者の設問別平均スコアを棒グラフで表示。

| スコア | 色 |
|--------|-----|
| 4.0以上 | 緑 |
| 3.0〜3.9 | 黄 |
| 3.0未満 | 赤 |

### 部署別比較（レーダーチャート）
部署ごとの8設問スコアをレーダーチャートで重ね描き。強み・弱みのパターンを部署間で比較できる。

## ファイル構成

```
10_employee_survey/
├── main.py          # サーベイ集計・設問別/部署別集計ロジック
├── gui.py           # Tkinter GUI（棒グラフ + レーダーチャート切り替え）
├── data/
│   └── survey.csv   # サンプルサーベイデータ（20名・5部署・8設問）
└── results/
    └── survey_report.csv   # 部署別集計出力
```

## サンプルデータ仕様（8設問・5段階評価）

| 列名 | 設問内容 |
|------|---------|
| Q1_業務満足度 | 現在の業務に満足しているか |
| Q2_職場環境 | 職場環境・設備は整っているか |
| Q3_上司との関係 | 上司との関係は良好か |
| Q4_給与待遇 | 給与・待遇に満足しているか |
| Q5_成長機会 | 成長できる機会があるか |
| Q6_会社への信頼 | 会社の方針・将来を信頼しているか |
| Q7_ワークライフバランス | 仕事と生活のバランスは取れているか |
| Q8_チームワーク | チームの連携は良いか |

## 主要ロジック（main.py）

| 関数 | 内容 |
|------|------|
| `load_survey(filepath)` | CSV読み込み・数値型変換 |
| `aggregate_by_question(df)` | 設問別の平均スコアを返す |
| `aggregate_by_department(df)` | 部署×設問の平均スコア表を返す |
| `get_overall_score(df)` | 全体の総合平均スコアを返す |

## 面接でのアピールポイント

- **人事の最新トピック：** エンゲージメントサーベイは多くの企業で毎年実施しており、「分析できる人事担当者」は希少
- `pandas` の `groupby` + `mean` で部署×設問のクロス集計を示せる
- `matplotlib` の極座標グラフ（レーダーチャート）という高度な可視化を説明できる
- 「スコアが低い設問・部署を自動でハイライトする」という分析視点を語れる
