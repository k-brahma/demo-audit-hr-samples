# 09 KPI トラッカー

## 概要

月次KPIの目標値・実績値を管理し、KPIごとの推移を折れ線グラフで表示するツール。達成率に応じて色分けし、未達KPIを一目で把握できる。

## 起動

```powershell
cd 09_kpi_tracker
python gui.py
```

## 使い方

1. 「CSVを開く」で `data/kpi.csv` を選択
2. 右下のドロップダウンでKPI名を選択すると折れ線グラフが更新される
3. 左下の「最新月サマリー」テーブルで全KPIの最新月達成状況を一覧確認
4. グラフの各データ点に達成率(%)が表示される
5. 「結果を保存」で `results/kpi_report.csv` に出力

## 達成率・ステータス分類

| 達成率 | ステータス | 表示 |
|--------|-----------|------|
| 90%以上 | 達成 🟢 | 緑 |
| 70〜89% | 警告 🟡 | 黄背景 |
| 70%未満 | 危険 🔴 | 赤背景 |

## グラフの見方

- **灰色破線：** 目標値の推移
- **緑の線：** 達成月の実績値
- **赤の線：** 未達月の実績値
- **各点の数値：** 達成率(%)

## ファイル構成

```
09_kpi_tracker/
├── main.py          # KPIデータ読み込み・達成率計算・サマリー集計ロジック
├── gui.py           # Tkinter GUI（Combobox + Treeview + matplotlib 折れ線グラフ）
├── data/
│   └── kpi.csv      # サンプルKPIデータ（3指標 × 6ヶ月）
└── results/
    └── kpi_report.csv   # 達成率計算済み出力
```

## サンプルデータ仕様

| 列名 | 内容 |
|------|------|
| KPI名 | KPIの名称（例: 売上高） |
| カテゴリ | 財務・営業・CS など |
| 年月 | YYYY-MM 形式 |
| 目標 | 目標値 |
| 実績 | 実績値 |

> 同じKPI名の行が月ごとに複数存在する縦持ちデータ形式。

## 主要ロジック（main.py）

| 関数 | 内容 |
|------|------|
| `load_kpi(filepath)` | CSV読み込み・数値型変換 |
| `calculate_achievement(df)` | 達成率・ステータス列を追加 |
| `get_kpi_names(df)` | KPI名一覧を返す |
| `get_kpi_trend(df, kpi_name)` | 指定KPIの時系列データを返す |
| `get_latest_summary(df)` | 最新月の全KPIサマリーを返す |

## 面接でのアピールポイント

- **経営管理・コントローラー職向け：** KPIのモニタリングは管理会計の基本業務として説明できる
- 「ピボットなしで縦持ちデータを時系列グラフにする」pandas操作を示せる
- `Combobox` によるインタラクティブなKPI切り替えUIを説明できる
- 線を月ごとにセグメント分割して達成・未達で色を変えるmatplotlibの高度な使い方を示せる
