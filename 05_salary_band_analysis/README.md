# 05 給与バンド分析ツール

## 概要

従業員の等級・部署・年収データを読み込み、ボックスプロット（箱ひげ図）で給与分布を可視化するツール。等級別・部署別の切り替え表示に対応。

## 起動

```powershell
cd 05_salary_band_analysis
python gui.py
```

## 使い方

1. 「CSVを開く」で `data/salary.csv` を選択
2. 左テーブルに等級別の最小・最大・平均・中央値・人数が表示される
3. 右側に箱ひげ図が描画される
4. 「表示グループ」を「等級別」↔「部署別」で切り替えられる
5. 「結果を保存」で `results/salary_stats.csv` に等級別統計を出力

## ボックスプロットの読み方

| 要素 | 意味 |
|------|------|
| 箱の下辺 | 第1四分位数（下位25%） |
| 箱の中線 | 中央値（50%） |
| 箱の上辺 | 第3四分位数（上位75%） |
| ひげ | 外れ値以外の最小・最大 |
| 点（◇） | 外れ値 |

## ファイル構成

```
05_salary_band_analysis/
├── main.py          # 給与統計計算・ボックスプロット用データ整形ロジック
├── gui.py           # Tkinter GUI（Treeview + matplotlib ボックスプロット）
├── data/
│   └── salary.csv   # サンプル給与データ（25名・5部署・等級2〜5）
└── results/
    └── salary_stats.csv   # 等級別統計出力
```

## サンプルデータ仕様

| 列名 | 内容 |
|------|------|
| 社員ID | 従業員識別子 |
| 部署 | 所属部署名 |
| 等級 | 職務等級（2〜5） |
| 年収 | 年間給与（円） |

## 主要ロジック（main.py）

| 関数 | 内容 |
|------|------|
| `load_salary(filepath)` | CSV読み込み・万円換算列を追加 |
| `get_grade_stats(df)` | 等級別の記述統計量を返す |
| `get_dept_stats(df)` | 部署別の記述統計量を返す |
| `get_boxplot_data(df, group_col)` | グループ別の年収リスト辞書を返す |

## 特徴・ポイント

- **報酬設計の実務知識：** 等級ごとの給与レンジ（バンド）管理は人事の基本業務であることを説明できる
- `pandas` の `groupby` + `agg` による複数統計量の同時計算を示せる
- `matplotlib` のボックスプロット（`boxplot`）とパッチ着色を説明できる
- 「給与の市場整合性チェックにこのグラフが使える」という業務ユースケースを語れる
