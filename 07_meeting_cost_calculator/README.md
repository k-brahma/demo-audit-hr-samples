# 07 会議コスト計算機

## 概要

参加人数と平均時給を設定してタイマーをスタートすると、会議コストがリアルタイムでカウントアップされるツール。コスト増加に応じて表示色が変化する。

## 起動

```powershell
cd 07_meeting_cost_calculator
python gui.py
```

## 使い方

1. 参加人数・平均時給を設定する（またはプリセットボタンを使用）
2. 「▶ 開始」でタイマーをスタート
3. 会議コストが0.5秒ごとにリアルタイム更新される
4. 「⏸ 一時停止」で停止、「▶ 再開」で再スタート
5. 「⏹ リセット」で0に戻す
6. 「💾 ログ保存」で会議記録を `results/meeting_log.jsonl` に追記保存

## 表示色の変化

| コスト | 色 |
|--------|-----|
| 1万円未満 | オレンジ |
| 1〜5万円 | 赤 |
| 5万円以上 | 濃い赤（警告） |

## プリセット

| プリセット名 | 人数 | 時給 | 想定時間 |
|------------|------|------|---------|
| 朝会（10分） | 8名 | ¥3,500 | 10分 |
| 週次MTG（60分） | 10名 | ¥4,000 | 60分 |
| 経営会議（120分） | 12名 | ¥8,000 | 120分 |

## コスト計算式

```
会議コスト（円）= 参加人数 × 平均時給 × 経過時間（時間）
```

## ファイル構成

```
07_meeting_cost_calculator/
├── main.py          # コスト計算・時刻フォーマット・ログ保存ロジック
├── gui.py           # Tkinter GUI（リアルタイムタイマー + コスト表示）
├── data/            # （このアプリはデータファイル不要）
└── results/
    └── meeting_log.jsonl   # 会議ログ（JSON Lines形式で追記）
```

## ログの形式（meeting_log.jsonl）

```json
{"日時": "2024-01-15 10:30:00", "参加人数": 10, "平均時給": 4000, "経過時間": "01:00:00", "コスト": 40000.0}
```

## 主要ロジック（main.py）

| 関数 | 内容 |
|------|------|
| `calculate_cost(participants, hourly_rate, elapsed_seconds)` | コストを計算して返す |
| `format_cost(cost)` | `¥40,000` 形式の文字列に変換 |
| `format_elapsed(seconds)` | `HH:MM:SS` 形式に変換 |
| `save_log(record)` | JSONL形式でログ追記 |

## 特徴・ポイント

- **最もインパクトのあるデモ：** 目の前でリアルタイムに数字が増えていく様子は誰でも直感的に理解できる
- Tkinter の `after()` メソッドによるノンブロッキングタイマー実装を説明できる
- 「コスト意識の可視化」という経営管理の観点から企画した課題設定を語れる
- プリセット機能・一時停止・ログ保存など、実用性を考えたUX設計を示せる
