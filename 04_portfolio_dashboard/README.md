# 04 株式ポートフォリオ 損益ダッシュボード

## 概要

保有株式の銘柄・保有数・平均取得単価・現在価格をCSVで管理し、損益額・損益率・資産配分を自動計算してダッシュボード表示するツール。

## 起動

```powershell
cd 04_portfolio_dashboard
python gui.py
```

## 使い方

1. 「CSVを開く」で `data/portfolio.csv` を選択
2. 上部サマリーに総取得額・総評価額・総損益額・損益率が表示される
3. 左テーブルに銘柄ごとの損益が一覧表示（含み益=赤、含み損=青）
4. 右円グラフで評価額ベースの資産配分を確認
5. 「結果を保存」で `results/portfolio_report.csv` に出力

## サマリーカードの指標

| 指標 | 説明 |
|------|------|
| 総取得額 | 保有数 × 平均取得単価 の合計 |
| 総評価額 | 保有数 × 現在価格 の合計 |
| 総損益額 | 総評価額 − 総取得額 |
| 損益率(%) | 総損益額 ÷ 総取得額 × 100 |

## ファイル構成

```
04_portfolio_dashboard/
├── main.py          # 損益計算・資産配分・サマリーロジック
├── gui.py           # Tkinter GUI（サマリーカード + Treeview + 円グラフ）
├── data/
│   └── portfolio.csv   # サンプルポートフォリオ（8銘柄）
└── results/
    └── portfolio_report.csv   # 損益計算済み出力
```

## サンプルデータ仕様

| 列名 | 内容 |
|------|------|
| ティッカー | 銘柄コード（例: 7203） |
| 銘柄名 | 会社名 |
| 保有数 | 保有株数 |
| 平均取得単価 | 取得時の平均単価（円） |
| 現在価格 | 現在の株価（円） |

> 現在価格は静的データ。実際の運用では `yfinance` 等で取得して列を更新する。

## 主要ロジック（main.py）

| 関数 | 内容 |
|------|------|
| `load_portfolio(filepath)` | CSV読み込み・数値型変換 |
| `calculate_pnl(df)` | 取得額・評価額・損益額・損益率・配分(%)を計算 |
| `get_summary(df)` | ポートフォリオ全体サマリー辞書を返す |

## 発展アイデア（面接でのトーク素材）

- `yfinance` で現在価格をリアルタイム取得する機能を追加できる（既に経験あり）
- 損切りラインの設定と通知機能に発展させられる
- 為替レートを加味した外国株管理にも応用できる

## 面接でのアピールポイント

- 自分の投資ツールとして実際に使うモチベーションのある題材
- `pandas` での複数列の同時計算（`evaluate`系処理）を示せる
- 「既存のスクリプトをGUI付きアプリに発展させた」という成長ストーリーが語れる
