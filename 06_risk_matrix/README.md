# 06 リスク評価マトリクス自動生成ツール

## 概要

リスクの「発生確率（1〜5）」×「影響度（1〜5）」の組み合わせで5×5のヒートマップを自動生成するツール。リスクの追加入力にも対応し、PNGファイルへの出力が可能。

## 起動

```powershell
cd 06_risk_matrix
python gui.py
```

## 使い方

1. 「CSVを開く」で `data/risks.csv` を選択
2. 右側に5×5のリスクマトリクス（ヒートマップ）が描画される
3. 左テーブルにリスク一覧がスコア降順で表示（高リスクは赤背景）
4. フォームに入力して「追加」ボタンでリスクを追加できる
5. 「PNG保存」で `results/risk_matrix.png` に画像出力
6. 「CSVエクスポート」で `results/risk_report.csv` に出力

## リスクレベル分類

| スコア（確率×影響度） | レベル |
|----------------------|--------|
| 16以上 | 高 🔴 |
| 8〜15 | 中 🟡 |
| 7以下 | 低 🟢 |

## ファイル構成

```
06_risk_matrix/
├── main.py          # リスクデータ読み込み・マトリクス構築・分類ロジック
├── gui.py           # Tkinter GUI（入力フォーム + Treeview + matplotlib ヒートマップ）
├── data/
│   └── risks.csv    # サンプルリスクデータ（12件・4カテゴリ）
└── results/
    ├── risk_report.csv     # リスク一覧CSV出力
    └── risk_matrix.png     # マトリクス画像出力
```

## サンプルデータ仕様

| 列名 | 内容 |
|------|------|
| リスクID | 識別子（例: R001） |
| 分類 | リスクカテゴリ（財務・情報・法務・人事・業務） |
| リスク内容 | リスクの説明 |
| 発生確率 | 1（非常に低い）〜5（非常に高い） |
| 影響度 | 1（非常に低い）〜5（非常に高い） |
| 担当者 | リスクオーナー部署 |

## 主要ロジック（main.py）

| 関数 | 内容 |
|------|------|
| `load_risks(filepath)` | CSV読み込み・スコア計算（確率×影響度） |
| `classify_risk_level(score)` | スコアを高・中・低に分類 |
| `build_heatmap_matrix(df)` | 5×5のnumpy配列（件数マトリクス）を生成 |
| `add_risk(df, ...)` | リスク行を追加して新DataFrameを返す |

## 面接でのアピールポイント

- **監査・リスク管理の実務知識：** COSO・J-SOXのリスク評価アプローチを知っているからこそ作れたツール
- 「手書きやExcelのマトリクスをコードで自動化した」という具体的な改善例として語れる
- `numpy` の2次元配列操作と `matplotlib` の `imshow` によるヒートマップを説明できる
- PNG保存機能により「報告書への貼り付けがワンクリックで完結」というユーザー目線の設計を示せる
